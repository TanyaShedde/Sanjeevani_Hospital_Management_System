# Sanjeevani Hospital Management System
A robust, full-stack, Server-Side Rendered (SSR) web application designed to manage complex hospital operations. This system handles interconnected records for patients, doctors, appointments, treatments, and billing.

## Tech Stack
* **Backend:** Python, Flask
* **Database:** MySQL (Relational schema with strict foreign key constraints)
* **Frontend:** HTML5, CSS3, Jinja2 Templating
* **Architecture:** Traditional MVC (Model-View-Controller) routing

## Key Features
* **Full CRUD Functionality:** Create, read, update, and delete records across all database tables.
* **Secure Data Flow:** Utilizes parameterized SQL queries via `mysql-connector-python` to mitigate SQL injection vulnerabilities.
* **Dynamic UI Rendering:** Employs Jinja2 to render server-side data directly into standard HTML/CSS.
* **Responsive Design:** Custom-built interface featuring an integrated state-based light/dark mode toggle.
